## .zshrc
```
export TERM="xterm-256color" # 設定256色輸出

ZSH_THEME="powerlevel9k/powerlevel9k"

POWERLEVEL9K_MODE='awesome-patched' ＃啟用漂亮icon
POWERLEVEL9K_LEFT_PROMPT_ELEMENTS=(dir dir_writable vcs) # 設定左方元素
```
