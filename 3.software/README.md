# 用途及其他設定

## Firefox 設定

### 滾輪捲動行數過少
`about:config`

搜尋

`mousewheel.min_line_scroll_amount`
設置為 45

### 網址列 click 全選
`browser.urlbar.clickSelectsAll`
set true
